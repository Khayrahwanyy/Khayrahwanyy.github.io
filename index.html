Portofolio Unggulan

Membuat Kabel LAN dan Fiber Optik

Fiber Optic

Pada gambar di atas, saya melakukan proses crimping dan penyambungan dua fiber optik. Proses crimping dilakukan dengan menggunakan alat khusus untuk menyambungkan kabel fiber optik ke konektor dengan memastikan bahwa serat optik terpasang dengan baik dan rapi. Setelah itu, saya melakukan penyambungan dua fiber optik dengan teknik splicing, yang melibatkan penggabungan ujung dua serat optik dengan hati-hati untuk memastikan transmisi sinyal yang optimal. Seluruh langkah ini dilakukan dengan ketelitian tinggi agar tidak ada gangguan pada kualitas sinyal yang diteruskan melalui fiber optik.

Membuat alat yang berhubungan dengan IoT

Internet Of Things

Pada gambar di atas, saya melakukan perencanaan pembuatan sistem IoT untuk timbangan dan berat badan bayi digital menggunakan platform Wokwi. Dalam perancangan ini, saya menggunakan komponen ESP32 sebagai mikrokontroler utama yang mengatur seluruh sistem, dilengkapi dengan sensor ultrasonik untuk mengukur tinggi badan bayi secara akurat. Untuk menampilkan hasil pengukuran, saya menggunakan LCD yang menampilkan berat badan bayi secara digital. Sistem ini dirancang agar dapat mengukur dan menampilkan data berat dan tinggi badan bayi secara real-time melalui website, memanfaatkan kemampuan ESP32 dalam menghubungkan perangkat dengan jaringan internet.

Menyetting mikrotik untuk pembuatan wifi

Network Engineer

Pada gambar di atas, saya melakukan penyettingan Mikrotik untuk pembuatan WiFi dengan mengkonfigurasi interface wireless sebagai access point, mengatur SSID dan keamanan jaringan, serta menambahkan DHCP server untuk distribusi IP otomatis. Setelah itu, saya memastikan koneksi internet dapat dibagikan melalui pengaturan NAT dan firewall. Semua pengaturan ini bertujuan untuk membuat jaringan WiFi yang stabil dan aman.
